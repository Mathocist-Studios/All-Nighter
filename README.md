## About The Project
All Nighter is a top-down maze game developed by Mathochist studios. The player must navigate through a series of mazes while avoiding obstacles and collecting items to progress to the next level. The game features multiple levels, each with increasing difficulty, and a variety of power-ups to help the player along the way.

## Getting Started
To get a local copy up and running follow these simple steps.

### Prerequisites
- Java Development Kit (JDK) 17 or higher
- Gradle
- An IDE such as IntelliJ IDEA or Eclipse
- Git

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/Mathocist-Studios/All-Nighter.git
    ```
   
2. Open the project in your IDE
3. Build the project using Gradle (Note: for desktop version, run the `lwjgl3:Tasks:application:run` task)
4. Run the application
5. Enjoy the game!
